# 🚀 Mi Portafolio Web - Flask Version

Este es mi espacio personal donde documento y presento mi trayectoria como desarrollador. El objetivo principal es centralizar todos mis proyectos, detallando las tecnologías utilizadas y los retos resueltos en cada uno.

## 🛠️ Tecnologías Utilizadas

Para el desarrollo de este portafolio, decidí utilizar un stack sólido basado en Python para la lógica y un frontend limpio:

* **Backend:** [Flask](https://flask.palletsprojects.com/) (Micro-framework de Python)
* **Template Engine:** Jinja2 (Para renderizado dinámico de contenido)
* **Frontend:** HTML5, CSS3 y JavaScript
* **Despliegue:** Render

---

## 📋 Características Principales

1. **Galería de Proyectos:** Una sección dedicada que muestra tarjetas con cada uno de mis trabajos.
2. **Detalles Técnicos:** Cada proyecto incluye una descripción detallada y una lista de etiquetas con las tecnologías empleadas.
3. **Diseño Responsive:** Adaptado para visualizarse correctamente en dispositivos móviles, tablets y computadoras.
